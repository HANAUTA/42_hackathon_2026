# 画面仕様：撮影・送信画面

---

## 撮影画面

### 起動時の動作

- 画面表示と同時にカメラ自動起動

### レイアウト

- カメラは画面内に収める（フルスクリーンではない）

### 機能

| 機能 | 内容 |
|------|------|
| 撮影 | 動画撮影の開始・停止 |
| カメラ切替 | インカメラ ↔ アウトカメラ |
| タイマー設定 | 撮影開始までのカウントダウン |

### 撮影後

- 送信画面へ遷移

---

## 送信画面

### 表示内容

- 撮影した動画のプレビュー

### 機能

| 機能 | 内容 |
|------|------|
| 取り消し | 撮影画面へ戻る（動画は破棄） |
| 送信 | 選択したグループへ動画を投稿 |

---

## 送信先選択

### 表示内容

参加中のグループを一覧表示する。

| 状態 | 表示 | 選択 |
|------|------|------|
| 送信可能 | 通常表示 | 選択可能 |
| 送信済み（同時間帯） | グレーアウト | 選択不可 |

- 同一グループへの投稿制限は1時間に1回（詳細は[投稿ルール](./投稿ルール.md)参照）
- 複数グループへの同時送信が可能

### 送信処理

1. 動画を Supabase Storage にアップロード
2. `posts` テーブルにレコード作成
3. `post_shares` テーブルに選択グループ分レコード作成
4. ホーム画面へ遷移
