# 画面遷移

## 遷移図

```
起動
 └── スプラッシュ画面
      ├── 未ログイン → ログイン画面
      │    └── 認証成功
      │         ├── 初回ログイン → プロフィール入力画面 → ホーム画面
      │         └── 2回目以降  → ホーム画面
      └── ログイン済み → ホーム画面
```

```
ホーム画面
 ├── [ログ作成ボタン] → 撮影画面 → 送信画面 → ホーム画面
 ├── [グループ作成ボタン] → グループ作成画面 → ホーム画面（自動参加）
 ├── [グループ参加ボタン] → グループ参加画面 → ホーム画面
 └── [グループ名タップ] → グループ詳細画面
```

---

## 画面一覧

| No | 画面名 | 主な用途 |
|----|--------|---------|
| 1 | スプラッシュ画面 | 起動時・ログイン状態確認 |
| 2 | ログイン画面 | 認証 |
| 3 | プロフィール入力画面 | 初回ユーザー情報登録 |
| 4 | ホーム画面 | 自分のVlog・グループ一覧 |
| 5 | グループ作成画面 | グループ作成・招待コード発行 |
| 6 | グループ参加画面 | 招待コードで参加 |
| 7 | グループ詳細画面 | グループ内投稿の時系列表示 |
| 8 | 撮影画面 | 動画撮影 |
| 9 | 送信画面 | 撮影動画のプレビュー・グループ選択・送信 |
