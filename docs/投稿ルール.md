# 投稿ルール

---

## Vlog（自分のタイムライン）

- 投稿数・頻度に制限なし

---

## グループへの投稿制限

- **1グループにつき、1時間に1回まで**
- 制限の単位は「時台（0〜23時）」で管理

### 例

| 時刻 | 操作 | 結果 |
|------|------|------|
| 11:30 | グループAへ投稿 | 成功 |
| 11:45 | グループAへ再投稿 | **不可**（11時台に投稿済み） |
| 11:45 | グループBへ投稿 | 成功（グループBは未投稿） |
| 12:05 | グループAへ投稿 | 成功（12時台は未投稿） |

---

## 実装上の判定ロジック

送信画面でグループの選択可否を判定する。

```
1時間に1回制限の判定:
  post_shares テーブルで
  group_id = 対象グループ
  AND shared_hour = 現在の時（hour）
  AND date(created_at) = 今日の日付
  のレコードが存在する → 送信済み → グレーアウト
```
