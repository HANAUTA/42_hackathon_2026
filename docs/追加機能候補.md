# 追加機能候補（学生向け）

MVP完成後に取り組む機能の候補。難易度別に整理。

---

## 初級

| 機能 | 概要 |
|------|------|
| いいね | 投稿にいいねをつける |
| プロフィール編集 | 名前・アイコンを変更する |

---

## 中級

| 機能 | 概要 |
|------|------|
| コメント | 投稿にコメントをつける |
| 投稿削除 | 自分の投稿を削除する |
| グループ脱退 | グループから退出する |

---

## 上級

| 機能 | 概要 |
|------|------|
| リアクション | 絵文字などのリアクション機能 |
| 通知 | 新着投稿のプッシュ通知 |
| 未読管理 | 未視聴の投稿をバッジ等で表示 |
