# 画面仕様：ホーム画面

---

## 概要

ログイン後のメイン画面。自分のVlog一覧とグループ一覧を表示する。

---

## 機能ボタン

| ボタン | 遷移先 |
|--------|--------|
| ログ作成 | 撮影画面 |
| グループ作成 | グループ作成画面 |
| グループ参加 | グループ参加画面 |

---

## 表示セクション

### 自分のVlog

- 自分が投稿した動画一覧
- 時系列順（新しい順）
- Supabase Storage の video_url から動画を表示

### グループ一覧

- 参加中のグループ名一覧
- グループ名をタップ → グループ詳細画面へ遷移
