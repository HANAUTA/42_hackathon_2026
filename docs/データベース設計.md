# データベース設計

## テーブル一覧

### users

| カラム | 型 | 説明 |
|--------|-----|------|
| id | uuid (PK) | Supabase AuthのユーザーID |
| name | text | 表示名 |
| icon_url | text | アイコン画像URL（Storage） |
| created_at | timestamp | 作成日時 |

---

### groups

| カラム | 型 | 説明 |
|--------|-----|------|
| id | uuid (PK) | グループID |
| name | text | グループ名 |
| invite_code | text (UNIQUE) | 招待コード（自動生成） |
| owner_id | uuid (FK: users.id) | 作成者 |
| created_at | timestamp | 作成日時 |

---

### group_members

| カラム | 型 | 説明 |
|--------|-----|------|
| id | uuid (PK) | |
| group_id | uuid (FK: groups.id) | |
| user_id | uuid (FK: users.id) | |
| joined_at | timestamp | 参加日時 |

- (group_id, user_id) にUNIQUE制約

---

### posts

| カラム | 型 | 説明 |
|--------|-----|------|
| id | uuid (PK) | 投稿ID |
| user_id | uuid (FK: users.id) | 投稿者 |
| video_url | text | Supabase StorageのURL |
| created_at | timestamp | 投稿日時 |

---

### post_shares

投稿と共有先グループを分離するテーブル。1投稿を複数グループに共有できる構造。

| カラム | 型 | 説明 |
|--------|-----|------|
| id | uuid (PK) | |
| post_id | uuid (FK: posts.id) | |
| group_id | uuid (FK: groups.id) | 共有先グループ |
| shared_date | date | 共有した日付（日本時間） |
| shared_hour | int | 共有時の時間帯（0〜23） |
| created_at | timestamp | 共有日時 |

- 同一グループへの同時間帯投稿制限は `UNIQUE (group_id, shared_date, shared_hour)` で管理
- ※ `date(created_at)` のような式インデックスはPostgresのIMMUTABLE制約で使えないため、日付は `shared_date` カラムとして明示的に保存する

---
