# 技術選定

## 技術スタック

| 分類 | 技術 | 用途 |
|------|------|------|
| フロントエンド | Flutter | UI・アプリ全体 |
| バックエンド | Supabase Authentication | ログイン・ユーザー管理 |
| バックエンド | Supabase Database (PostgreSQL) | データ永続化 |
| バックエンド | Supabase Storage | 動画ファイル保存 |

---

## Flutter パッケージ候補

| パッケージ | 用途 |
|-----------|------|
| `camera` | 動画撮影 |
| `video_player` | 動画再生・プレビュー |
| `supabase_flutter` | Supabase連携 |
| `google_sign_in` | Googleログイン |

---

## 対応プラットフォーム

- **Android**（メイン）
- **Flutter Web**（開発中の動作確認用）

---

## 動画処理方針

- 動画圧縮なし
- 撮影 → Supabase Storageへアップロード → URLをDBに保存 → 一覧表示
