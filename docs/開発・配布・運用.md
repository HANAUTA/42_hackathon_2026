# 開発・配布・運用

---

## 開発環境

- Flutter（Android対応 / Flutter Web対応）
- Supabase（クラウドホスト）

---

## 開発中の動作確認

Flutter Webで確認する。

```bash
flutter run -d chrome
```

---

## APK配布

### ビルド

GitHub ActionsでAPKを自動生成する。

```yaml
# .github/workflows/build.yml（例）
- name: Build APK
  run: flutter build apk --release
```

### インストール

生成したAPKをAndroid実機にインストールして確認する。
