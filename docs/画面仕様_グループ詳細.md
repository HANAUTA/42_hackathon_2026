# 画面仕様：グループ詳細画面

---

## 概要

グループ内の投稿を時系列で閲覧できる画面。

---

## 初期表示

- 現在時刻の時間帯の投稿を表示する

例：現在 11:40 → 11時台（11:00〜11:59）の投稿を表示

---

## 時間移動

左右タップで1時間単位で移動する。

| 操作 | 動作 |
|------|------|
| 左タップ | 1時間前の投稿へ |
| 右タップ | 1時間後の投稿へ |

```
例：11時 → 左タップ → 10時 → 左タップ → 9時
```

---

## 日付移動

左右スワイプで1日単位で移動する。

| 操作 | 動作 |
|------|------|
| 左スワイプ | 前日の投稿へ |
| 右スワイプ | 翌日の投稿へ |

```
例：今日 → 左スワイプ → 昨日 → 左スワイプ → 一昨日
```

---

## その他の表示・機能

| 要素 | 内容 |
|------|------|
| メンバー一覧 | グループに参加中のメンバー表示 |
| 招待コード表示 | グループの招待コードを確認できる |
| グループ退出 | グループから退出できる |
