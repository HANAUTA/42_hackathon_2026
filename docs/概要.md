# Setlog風アプリ 概要

## プロジェクト概要

42ハッカソン向けに開発する、Setlogライクな動画共有アプリのMVP。

---

## MVP機能一覧

| カテゴリ | 機能 |
|--------|------|
| 認証 | メールログイン、Googleログイン |
| ホーム | 自分のVlog一覧、所属グループ一覧 |
| グループ | グループ作成、招待コード発行、招待コードで参加 |
| 投稿 | 動画撮影、共有先グループ選択、動画投稿 |
| グループ詳細 | グループ内の投稿一覧表示 |
