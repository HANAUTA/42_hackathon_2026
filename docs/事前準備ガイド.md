# 事前準備ガイド

ハッカソン当日にスムーズにスタートするために、**以下の5つを事前に準備**しておいてください。
すべて無料です。当日は Flutter のインストールから始めるので、ここまでできていれば大丈夫です。

> 所要時間の目安：**15〜30分**（すでにいくつか入っている人はもっと早いです）

---

## ✅ 1. パソコン（Windows または Mac）

- **Windows 10 以降** または **macOS** が必要です
- 空き容量は **5GB 以上**あると安心です（Flutter SDK + 開発ツール分）
- Chromebook・iPad は対象外です

---

## ✅ 2. Google Chrome（ブラウザ）

開発中のアプリを Chrome 上で動かして確認します。**開発のメインツール**です。

### インストール方法

1. [google.com/chrome](https://www.google.com/chrome/) を開く
2. 「Chrome をダウンロード」→ インストール

> すでに入っている人はそのままでOKです。

---

## ✅ 3. VS Code（エディタ）

コードを書いたり、ファイルを編集したりするためのアプリです。

### インストール方法

1. [VS Code 公式ページ](https://code.visualstudio.com/) を開く
2. 自分のパソコン（Windows / Mac）に合わせてダウンロード → インストール
3. インストールしたら一度開いて、起動できることを確認

### おすすめ：Flutter 拡張機能を入れておく（任意）

1. VS Code を開く
2. 左の四角いアイコン（拡張機能）をクリック
3. 検索欄に **「Flutter」** と入力
4. 一番上に出てくる **「Flutter」**（Dart Code 作）を **「Install」**

> コードの補完やエラー表示が効くようになるので、入れておくと開発がぐっと楽になります。

---

## ✅ 4. Git（バージョン管理ツール）

コードのダウンロード（clone）や変更の保存（commit）・共有（push）に使います。

### 入っているか確認する

ターミナル（Mac）または PowerShell（Windows）を開いて、次を打ちます。

```bash
git --version
```

バージョンが表示されれば入っています。入っていなければ↓

### 🪟 Windows の人

1. [git-scm.com/download/win](https://git-scm.com/download/win) を開く
2. **「Click here to download」** をクリック → インストーラを実行
3. 設定はすべて **デフォルトのまま「Next」** を押し続けてOK
4. インストール後、**PowerShell を開き直して** `git --version` で確認

### 🍎 Mac の人

ターミナルで次を実行します。

```bash
git --version
```

初めて実行すると「Command Line Tools をインストールしますか？」とポップアップが出ることがあります。
**「インストール」を押して**ください（数分かかります）。終わったらもう一度 `git --version` で確認。

> Homebrew が入っている人は `brew install git` でもOKです。

---

## ✅ 5. GitHub アカウント

コードの保存・共有に使います。当日チームのリポジトリに招待するので、**アカウントが必要**です。

### アカウントを作る

1. [github.com](https://github.com) を開く
2. **「Sign up」** をクリック
3. メールアドレス・パスワード・ユーザー名を入力して登録
4. メールで届く確認コードを入力して完了

> すでにアカウントがある人はそのままでOKです。
> 当日、**GitHub のユーザー名**をチームメンバーに共有してもらうので、覚えておいてください。
